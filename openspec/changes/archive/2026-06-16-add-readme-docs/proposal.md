## Why

The project has no documentation. Users discovering `parse-figma-canvas` need a clear README explaining what the tool does, how to install it, and how to use each command — especially the new `-o` and `all` features. Bilingual docs (English + Cantonese) serve the local dev community better.

## What Changes

- Add `README.md` (English) with: project background, installation, usage examples for every command, `-o` flag explanation, `all` command explanation
- Add `README-HK.md` (Cantonese with emoji) covering the same content in 廣東話

## Capabilities

### New Capabilities

- `readme-english`: English README with background, installation, full usage examples for all commands
- `readme-cantonese`: Cantonese README (with emoji) covering same content

### Modified Capabilities

(none)

## Impact

- **Files**: New files `README.md` and `README-HK.md` at project root
- **Code**: No code changes
- **Dependencies**: None
